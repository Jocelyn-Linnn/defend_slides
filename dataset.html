01
Test Set Construction
MIMIC-CXR-JPG · Frontal, Single-Study Test Images
~227K chest X-ray image–report pairs (Beth Israel Deaconess Medical Center) — a widely used public benchmark. Only the official test split is used, to avoid overlap with MedGemma's pretraining data
1
Keep frontal views only (AP/PA) — MedGemma is designed for frontal chest X-rays
2
Regex-filter out reports with comparison language (e.g. "compared to prior", "interval change", "stable") — these depend on prior-image context unavailable to a single-image model
3
Randomly sample from the remaining qualifying images
→ n = 500 test images
02
Sample-Size Validation
Bootstrap 95% CI Confirms Stability
Evaluating all 220K+ images is computationally infeasible — each sample requires multiple LLM inference passes (MedGemma generation + GREEN evaluation). 500 was chosen and then validated post hoc
Bootstrap 95% CI · 1000 resamples (n=500)
RadGraph F1 (partial)0.286 ± 0.012
GREEN0.382 ± 0.020
CheXBert Micro F10.457 ± 0.031
CheXBert Macro F10.344 ± 0.027
All metrics converge by ~300 samples; worst-case CI half-width is only ±0.044

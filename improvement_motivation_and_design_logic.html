The Problem
Supplementing Omissions Inevitably Adds Hallucinations
The core pipeline's design direction is "supplement what's missing" — retrieval + QA add clinical findings the initial report omitted. But this same mechanism also introduces QA-bias hallucinations

The Design Logic
→ A Complementary Fact-Checking Examiner
Mahmood et al.'s image-driven Fact-Checking Examiner works in the opposite direction: given a report + its image, classify each sentence as real/fake and delete the fake ones
1
Jointly encode the image and each report sentence into a shared embedding space
2
A binary classifier scores each sentence's P(Real)
3
Sentences below the threshold are removed
Two-stage framework: this system supplements omissions, the Examiner removes fabrications — together balancing false negatives and false positives

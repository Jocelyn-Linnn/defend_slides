Failure Mode Analysis of Metric Regression
Pattern 1
Inaccurate QA Answers Introduce Hallucinations
QA can hallucinate answers even when the initial report is correct
Example
GT"The cardiomediastinal silhouette is normal" — no cardiomegaly
MedGemmaAlready correct — clear lungs, normal heart size
QA StepAsked about reduced heart size → answered "Yes... suggesting some resolution of cardiomegaly" — pure hallucination, since no cardiomegaly ever existed
Final ReportIncorporates the fabricated QA answer, introducing a finding absent from both GT and MedGemma's baseline
Pattern 2
Hallucinations Compounding on a Flawed Baseline
QA tends to confirm more errors on top of an already-wrong baseline
Example
GTTube position too high; pacemaker/catheter unchanged; lungs otherwise unchanged
MedGemmaAlready wrong — falsely lists 4 findings: bilateral (R+L) pleural effusion and bilateral lower-lobe atelectasis
QA StepBuilt on that false premise → "lingular atelectasis?" → Yes; "signs of recent CABG?" → Yes (sternal wires present, but GT mentions no recent surgery)
Final ReportStacks every affirmative answer → 6+ findings inconsistent with GT
GREENDrops to 0.222

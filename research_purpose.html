Chest X-ray Report Refinement System Research Purpose
Research Objectives
Integrated Report Generation & Refinement Pipeline MedGemma as the base VLM with modular design — any medical VLM supporting report generation and visual QA can be plugged in.
Radiology Association Graph Construction Build a compact clinical graph from RadGraph via node merging, relation abstraction, and structure compression for dense semantic representation.
Graph Retrieval Module for Clinical Associations Retrieve key nodes and latent clinical relations from the graph; compare different retrieval strategies on clinical reasoning depth.
LLM-based Clinical QA & Report Refinement Convert retrieved associations to clinical questions → interactive confirmation with MedGemma → LLM integrates answers into the final refined report.
